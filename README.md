# BATYR

> *A brutal cinematic action experience inspired by Kazakh mythology and nomadic warrior culture.*

---

# 🎮 About The Project

**BATYR** is a first-person cinematic melee action game where the player defends their homeland against possessed invaders using a sword and shield.

Inspired by games like *Sifu*, *Sekiro*, and *Ghost of Tsushima*, BATYR focuses on tactical combat, impactful strikes, and immersive atmosphere rooted in Central Asian culture and mythology.

The project combines:

* Story-driven gameplay
* Cinematic presentation
* Immersive environments
* Emotional dialogue and character development
* Dynamic action sequences

The goal is to create an experience that feels like playing through an interactive movie.
